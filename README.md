# Proyecto Lights Out - Álgebra Aplicada 2025
Este proyecto implementa en Python el modelo matemático del juego Lights Out, utilizando álgebra lineal sobre el conjunto {0,1}.
El archivo principal, lights_out.py, recibe un tablero inicial y calcula automáticamente el vector solución que indica qué luces deben presionarse para apagar todo el tablero.

## ¿Cómo correr el archivo?
1. **Tener instalado Python**
   
   Esto se puede comprobar escribiendo
   ```
   python --version
   ```
   o
   ```
   python3 --version
   ```
   
2. **Instalar la librería utilizada (NumPy)**
   
   Para instalarla, escrbir en la terminal
   ```
   pip install numpy
   ```
   o
   ```
   pip3 install numpy
   ```
   
3. **Guardar el archivo ```lights_out.py```**

   Guardar el archivo ```lights_out.py``` en la carpeta que desees.

   En la terminal, desplazarse hasta esa carpeta utilizando el comando ```cd```.
   
4. **Ejecutar el programa**

   Una vez ubicados en la carpeta donde se encuentra el archivo, ejecutar
   ```
   python lights_out.py
   ```
   o
   ```
   python3 lights_out.py
   ```
